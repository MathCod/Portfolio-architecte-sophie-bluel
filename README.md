---

## 📋 Présentation du projet
L'objectif était de rendre dynamique le site statique de Sophie Bluel. J'ai pris en charge la récupération des données via une API, la création de filtres de navigation, et l'implémentation d'une interface d'administration sécurisée permettant la gestion des travaux (ajout/suppression).

---

## 🛠️ Compétences validées

- **Manipulation du DOM :** Génération dynamique des projets de la galerie sans rechargement de page.
- **Appels API (Fetch) :** Récupération et envoi de données (JSON) vers le back-end.
- **Système de Filtrage :** Création de filtres par catégories d'architecture.
- **Authentification :** Mise en place d'une page de login et gestion du stockage sécurisé du token (sessionStorage).
- **Modale Interactive :** Conception d'une interface modale à deux niveaux (visualisation et ajout de nouveaux projets).
- **UX/UI :** Gestion des retours utilisateurs lors de l'envoi de formulaires (prévisualisation d'image, messages d'erreur).

---

## 🚀 Installation

### 1. Pré-requis
Le projet nécessite le lancement d'un **Back-end local** (fourni par OpenClassrooms).

### 2. Lancer le Back-end
```bash
# Se rendre dans le dossier backend
cd backend
npm install
npm start
# Le serveur tourne par défaut sur le port 5678.
```
### 3. Lancer le Front-end
Le front-end est en HTML/CSS/JS pur. Il suffit d'ouvrir le fichier index.html avec une extension type Live Server (VS Code) ou simplement de l'ouvrir dans votre navigateur.
